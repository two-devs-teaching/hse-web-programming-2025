# Задание 3. Браузерное API

**Задание:** интегрировать в свой веб-проект как минимум одно браузерное API.

Для вдохновения и выбора подходящего API можно ознакомиться со списком доступных браузерных API на [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/API).

**Критерии выполненного задания:**

* Есть ссылка на код проекта с интегрированным браузерным API
* Выбранное API используется осмысленно и улучшает пользовательский опыт
* Код структурирован и разделён на логические модули
* Обработаны основные сценарии использования API
* Добавлена обработка ошибок для случаев, когда API недоступно
* Проект корректно работает в современных браузерах

**Примеры популярных браузерных API:**
* Geolocation API — для работы с геолокацией
* Web Speech API — для распознавания и синтеза речи  
* Intersection Observer API — для отслеживания видимости элементов
* Local Storage / Session Storage — для хранения данных
* Notification API — для показа уведомлений
* Camera API (getUserMedia) — для работы с камерой
* Web Workers — для выполнения задач в фоновом режиме
* Drag and Drop API — для реализации перетаскивания элементов
